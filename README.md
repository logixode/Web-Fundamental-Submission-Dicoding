## Prerequisite

PurgeCSS

```sh
npm install purgecss --global
```

or

```sh
yarn global add purgecss
```

## Installation

This step will use PurgeCSS to remove unused css from the project

NPM :

```sh
npm run build
```

Yarn :

```sh
yarn build
```
